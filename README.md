# 📊 PlannerPro

Une application de gestion de tâches professionnelle développée avec **Streamlit** et **Python**.

## ✨ Fonctionnalités

- ✅ Ajout de tâches
- 📂 Catégories (day / week / month)
- ✔️ Marquer comme fait / pas fait
- 🗑️ Supprimer des tâches
- 📊 Statistiques en temps réel
- 📱 Interface moderne et responsive

## 🚀 Installation

bash
# Cloner le dépôt
git clone https://github.com/stephanieleurquin/Planner-public.git

# Aller dans le dossier
cd Planner-public

# Installer les dépendances
pip install -r requirements.txt

# Lancer l'application
streamlit run app.py
